# Tic-Tac-Toe Game

## Overview
Tic Tac Toe is a responsive web-based game built with HTML, CSS, and JavaScript. The game allows two players to compete by taking turns placing X and O on a 3×3 grid. The first player to align three matching symbols horizontally, vertically, or diagonally wins the game.

The project focuses on implementing game logic using JavaScript while providing a clean and interactive user interface.

 ## Features
 
- Smart computer opponent using Minimax.
- Beautiful UI with smooth transitions.
- Two-player gameplay (X vs O).
- Interactive 3×3 game board.
- Automatic winner detection.
- Detects draw situations.
- Restart the game without refreshing the page.
- Responsive design for desktop and mobile devices.
- Simple and modern user interface.


## Game Modes
The game offers two different playing modes:

- **Single Player:** Play against the computer.
- **Two Players:** Play locally with another player on the same device.

## Game Rules
- Two players take turns placing X and O.
- The first player to get three symbols in a row wins.
- If all cells are filled without a winner, the game ends in a draw.
- Players can restart the game at any time.
